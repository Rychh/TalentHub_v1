# TalentHub_v1

How to install TalentHub locally:
- install python 3.7.2
- install heroku CLI
- clone repo and install requirements
```
git clone https://github.com/Rychh/TalentHub_v1
pip install -r requirements.txt
```
- complete setup
```
chmod +x setup.sh
./setup.sh
```
- run heroku localy 
``` 
heroku local web 
```
